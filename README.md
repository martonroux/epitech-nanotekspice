## About the project

NanoTekSpice is a tek2 project that aims to create a logic simulator software in C++

### Built with

* [C++](https://en.wikipedia.org/wiki/C%2B%2B)
* [Makefile](https://en.wikipedia.org/wiki/Makefile)


## Getting started

### Prerequisites

* make
* g++

## Installation

1. Clone the repo
   ```sh
   git clone git@github.com:EpitechPromo2027/B-OOP-400-MPL-4-1-tekspice-marton.roux.git
   
2. Build the project
   ```sh
   make
   ```

3. Run the project
   ```sh
    ./nanotekspice [file.nts]
    ```
